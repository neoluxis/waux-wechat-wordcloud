# 微信词云

## 简介

为微信接入消息分析算法，生成词云图片，实现消息记录分析。

## 功能

1. 分析群聊、好友的聊天记录生成词云
2. 在好友中可以合并双方，也可以分别展示词云
3. 可以设置分析时间，分析时间段内的消息记录。时间默认为过去一天
4. 支持命令触发，经过授权的用户或群消息可以触发
5. 关键词过滤，在分析中过滤该词
6. 所有计算在本地完成，保证隐私

支持 WAuxiliary 1.2.7版本

## 用法

1. 将zip文件解压至/Android/media/com.tencent.mm/WAuxiliary/Plugin/
2. 在 WAuxiliary 中打开插件开关
3. 在需要支持词云的聊天中运行命令 `,wcenable`

## 命令

**激活**

在需要支持词云的聊天中运行命令 可以激活

```
,wcenable
，wcenable
/wcenable
```

输入命令可以取消激活

```
,wcdisable
，wcdisable
/wcdisable
```

取消激活后可以清空该聊天的缓存

```
,wcclean
，wcclean
/wcclean
```

可以使用命令查看状态

```
,wcstatus
，wcstatus
/wcstatus
```



**触发**

在输入栏输入命令，或者收到这样的消息即可直接触发

```
,wc -t [time] -f [filter1] -f [filter2]
，wc -t [time] -f [filter1] -f [filter2] # 中文逗号
/wc -t [time] -f [filter1] -f [filter2]
```

输入以下命令可以在 UI 中配置时间和关键词过滤

```
,wcm
，wcm # 中文逗号
/wcm
```

---

**时间格式**

可以直接使用 `-` 连接两个时间，`t` 开头表示今天的 24 小时制时间，`d` 开头表示日期或日期时间，该配置前后位数、配置方法需要保持统一。支持的时间格式有为 `<t>hhmm` , `<d>MMdd`, `<d>yyyyMMdd`, `<d>yyyyMMddhhmm`

```
t1200-1300 # 今天 12:00 ～ 13:00
d0326-0327 # 本年度3月26日到3月27日
d20260326-20260327 # 2026年3月26日0时到3月27日0时
d202603261330-202603270000 # 2026年3月26日13时30分到3月27日0时
```

也可以直接传入一个时间，表示从那个时间至今

```
t1230 # 从12点30分到现在
d20260326 # 从 2026年3月26日0时到现在
d202603261330 # 2026年3月26日13时30分到现在
```

还可以直接 `-` 开头，计算从现在开始回溯的时间。支持`月`，`周`，`天`， `小时`， `分钟` 

```
-1M # 过去一个月
-1w # 过去一周
-3d # 过去3天
-2h # 过去2小时
-30min # 过去半小时
```

不传入时间，默认分析过去一天的消息记录。

**虽然会使用多线程等优化，但是仍然不建议一次性分析过多聊天记录，以防卡顿等问题。**
